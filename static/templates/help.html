FAQs

This list should be made into a contracted list. Clike an arrow -> more contend appears, which are hidden by default.

1. What is DBGC used for?

thermodynamic properties prediction

Currently This is just an illustraction for DBGC method. Only enthalpy for alkanes and alkenes, and their radical are supported. In the future, we will try to include entropy and heat capacity prediction.

2. what is group additivity (GA)?

Benson group THERM combustion mechnism

we are trying to develop a automotic group contribution method, although it is not sinple additivity.

3. What's the advantages of DBGC method?

Automatic, systematic, and adaptive.

4. How to use this DBGC web app?

Curretly two types of input formats are supported. One is upload the .gjf file. The other is to enter the geometry directly. In the future, we will support the input to connectivity.

5. How does DBGC work?

distance-based, two parts, more details see the paper

Training test set. Regression and ANN. code can be found at .github.

6. How to convert the geometry or connectivity into DBGC vectors?

recognize the group, count the bonds, transfer with exp(-3dij). The code can be found in at .github.

7. What is used as the database currently?

the database consisted of C3-C13 alkanes C3-C10 alkenes C2-C9 alkenyl radicals C3-C10 alkyl radicals. The database can be downloaded at .github. A gui will be provided on this site to view the database.

8. How to construct my own database?

collect the experimental data, however limited. Calculation is recommended. Our vision, computation is becoming cheaper and cheaper. We have relatively accurate theoretical methods. Assistant code at .github for conformational analysis and quantum computation result extraction.

10. Is this an open-source program?

Certainly! Open-source lisence. The source code can be downloaded at .github.
